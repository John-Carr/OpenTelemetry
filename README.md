This project is 3 main parts: backend-http, backend-sockets, client
## backend-http
This is for all of our standard CRUD updates.
* Creating vehicles
* Creating telemetry items
* Getting historical data
Hosted on port 5000
## backend-sockets
This is for all the realitme stuff.
* Live telemetry
Hosted on port 4000
